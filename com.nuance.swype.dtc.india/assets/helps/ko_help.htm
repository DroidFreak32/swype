Swype로 네 가지 입력모드인 Swype, 말하기, 쓰기, 또는 탭을 매끄럽게 이동할 수 있습니다.

  • 스와이프

    Swype 은 새로운 입력 방식입니다 글자에서 손가락을 떼지 않은 채 미끄러지듯 움직여 단어를 입력할 수 있습니다. 단어 첫 글자에 손가락을 대고 글자에서 글자로 그림을 그리듯 손가락을 움직인 뒤 마지막 글자에서 손가락을 뗍니다. 필요한 경우 띄어쓰기가 자동으로 입력됩니다.

    자세히 알아보기
    • Swype 키

      스와이프 키는 스와이프 로고가 있는 키입니다. Swype 설정을 보려면 Swype 키를 계속해서 누릅니다

      설정 키는 보다 빠른 입력을 도와주는 동작을 실행할 때도 사용됩니다.

    • 단어선택

      트레이스 또는 탭을 반복하면 단어 선택 목록의 추천단어를 선택하여 사용합니다. 또는 단어 선택창을 손가락으로 내려가면서 원하는 단어를 선택할 수도 있습니다. 시스템 이용 서비스인 터치하여 탐색을 사용할 경우 이 기능은 이용하실 수 없습니다.

      목록에서 텍스트를 선택하려면 손가락으로 원을 그려 목록의 항목을 들을 수 있습니다. 시계 방향으로 원을 그리면 목록의 처음부터 시작하고 시계 반대 방향으로 원을 그리면 목록의 끝부터 시작합니다. 가장 마지막에 들린 항목을 선택하려면 손가락을 들어 올리세요. 단어 선택 목록에서 단어를 선택하려면 목록의 첫 번째 단어가 들릴 때까지 키보드에서 손가락을 들어 올립니다. 그런 다음 원형을 그리는 동작을 취합니다. 시스템 이용 서비스인 터치하여 탐색을 사용할 경우에만 이 기능을 이용하실 수 있습니다.

    • 글자 사이 오고 가기

      처음 입력하려는 단어가 Swype 의 추천 단어와 맞지 않을 때가 있습니다.

      예를 들어, “넌”과 “난”은 같은 움직임을 통해 입력됩니다. 그러나 글자 사이를 직선으로 움직일 필요는 없습니다. “ㅓ”를 피하며 손가락을 “ㄴ” 받침으로 트레이스할 때 단어선택 목록에 “난”이 제일 먼저 나타나는 것을 확인하십시오.

    • 다른 글자

      키를 길게 누르면 %, @와(과) 같은 기호 및 숫자 등 그 키에 해당하는 대체 문자 목록이 나타납니다.

      심볼 키(?123)를 치면 심볼 키보드로 이동합니다.

      주사용 키보드의 모든 문자는 화면 표시 여부에 상관 없이 트레이스할 수 있습니다. 키보드의 이 모드에서도 트레이스를 할 수 있지만, 적어도 번호 하나 또는 하나의 특수문자를 포함한 단어만 검색될 것입니다.

    • 단어 추가 및 삭제

      Swype는 사용하는 새 단어를 지능적으로 개인 사전에 추가시킵니다.

      단어를 하이라이트하고 를 쳐서 단어를 추가할 수도 있습니다. 추가하려는 단어를 탭해서 단어가 목록에 나타나도록 합니다.

      단어를 사전에서 삭제하려면 단어선택 목록에 있는 단어를 길게 누르세요. 시스템 이용 서비스인 터치하여 탐색을 사용할 경우 이 기능은 이용하실 수 없습니다.

    • 개인 설정

      Swype는 트위터, 지메일에서 신속히 단어를 사전에 추가시킬 수 있습니다. Swype 사용자 지정:

      1. 를 길게 누르십시오.
      2. Swype 설정 메뉴에서 내 단어 > 개인 설정을 선택합니다.
      3. 개인 설정 옵션에서 선택하여 안내에 따라 인증내용을 입력하십시오.
      4. 하나 또는 모든 소스에서 Swype 사용자 지정을 할 수 있습니다.
  • 말하기

    문자메시지와 이메일을 포함하여 페이스북과 트위터 업데이트까지 모든 문자로 된 내용을 음성으로 입력할 수 있습니다.

    자세히 알아보기
    • 문장부호

      손으로 문장부호를 입력하지 않아도 됩니다. 원하는 문장부호를 말하고 계속 입력하십시오. 시험해 보기:

      1. 음성 키를 누르고 말하십시오.
      2. 말할 내용: 저녁이 맛있었어요 느낌표
      3. 결과: 저녁이 맛있었어요!

    • 일부 키보드는 음성 입력을 지원하지 않습니다．

  • 쓰기

    글자와 단어를 부드럽게 끌어당기는 데 손가락을 사용하며 Swype는 그러한 움직임을 텍스트로 바꿉니다. 왼쪽에서 오른쪽으로 또는 겹쳐서 글자를 부드럽게 끌어당길 수 있습니다.

    자세히 알아보기
    • 필기 입력 사용
      1. 를 길게 누르기 한 후, 필기 입력 아이콘 쪽으로 손가락을 미십시오.
      2. 필기 입력 부분에 손가락으로 글자를 그리십시오.

    • 일부 키보드는 필기 입력을 지원하지 않습니다.

      시스템 이용 서비스인 터치하여 탐색을 사용할 경우 이 기능은 이용하실 수 없습니다.

  • 탭

    전통적인 수동 키보드 입력 방식입니다. Swype 키보드에서 탭 입력은 몇 가지 유용한 기능으로 더욱 쉬우면서도 효율적입니다.

    자세히 알아보기
    • 입력 오류 수정

      각각의 글자를 정확하게 치지 않아도 됩니다. 아는 만큼만 입력하면 Swype가 지능적으로 단어 제안을 합니다.

    • 단어 완성

      Swype는 글자 몇 개만 쳐도 단어를 추측해냅니다.

  • 언어

    키보드 언어전환: 스페이스 바를 길게 누르십시오. 팝업메뉴에서 원하는 언어를 선택하십시오.

    자세히 알아보기
    • “영” 키

      “영” 키를 길게 눌러 팝업 메뉴에서 언어를 변경할 수 있습니다.

      “영” 키를 짧게 눌러 한국어와 영어 사이에 레이아웃을 토글할 수 있습니다.

    • 키보드 배치 바꾸기

      한국어 멀티 글자 /필기 입력 모드를 위한 다양한 키보드 배치를 지원합니다. 두 손가락을 미끄러뜨리는 동작으로 키보드 배치를 바꾸거나 Swype 설정으로 들어가 키보드 배치를 원하는대로 바꿀 수 있습니다.

      키보드 입력 모드에서 두 손가락을 왼쪽에서 오른쪽으로 또는 오른쪽에서 왼쪽으로 미끄러트리면 전체 키보드에서 디지털 키패드 화면으로 전환시킬 수 있습니다.

      필기 확인 부분에서 두 손가락을 위쪽으로 또는 아래쪽으로 미끄러뜨려 전체화면과 부분 화면을 전환합니다.

    • 입력 키보드 닫기

      두 손가락을 아래쪽으로 미끄러뜨려 입력 키보드 창을 빠르게 닫을 수 있습니다.

    • 초성 검색

      각 음절의 초성 자음만을 탭하여 빠르게 한국어 단어를 입력할 수 있습니다. 모음이나 종성 자음을 더하면 예상 단어가 더욱 정확해집니다. 본 기능은 설정 > 한국어 > 키보드에서 사용 여부를 설정할 수 있습니다.

  • Swype Connect

    Swype Connect는 업데이트와 강력한 기능을 바로 귀하의 기기에 제공할 수 있게 합니다! Swype Connect는 3G 이상에서 작동하지만, 항상 Wi-Fi 연결을 추천합니다.

    자세히 알아보기
    • 언어 다운로드

      Swype에 추가 언어를 덧붙이기는 쉽습니다:

      1. 를 길게 누르고 언어 옵션을 선택하십시오.
      2. 언어 옵션 메뉴에서 언어 다운로드를 선택하십시오.
      3. 언어를 클릭하면 다운로드가 자동으로 시작됩니다.

    • Swype Connect는 모든 키보드에서 가능하지는 않습니다.
