• Isyarat Swype

    Isyarat Swype iku trabasan ana ing keyboard kanggo ngrampungi pakaryan kanthi cepet.


    Fitur iki ora kasedhiya yen sistem layanan aksesibilitas Explore-by-Touch mati.


    • Nggunakake Keyboard Edit

      Kanggo nggunakake Keyboard Edit, Swype saka menyang tombol Simbol (?123) ana ing keyboard.

    • Nggunakake keyboard angka

      Kanggo nggunakake Keyboard angka kanthi cepet, Swype saka menyang angka 5.

    • Ndhelikake keyboard

      Kanggo ndhelikake keyboard kanthi cepet, cukup Swype saka tombol Swype menyang tombol Backspace.

    • Mateni spasi otomatis

      Nonaktifake spasi otomatis sadurunge tembung sabanjure kanthi nge-Swype saka tombol Spasi menyang tombol Backspace.

    • Ngedit huruf gedhe-cilik tembung

      Owahi huruf gedhe-cilik tembung sauwise sampeyan ngetik kanthi nutul tembung, banjur nge-swype saka menyang tombol Shift . Dhaftar Pilihan Tembung kanthi opsi kapitalisasi bakal ditampilake, ngidini sampeyan kanggo milih aksara cilik, Dikapitalisasi, utawa KAPITAL KABEH.

    • Tandha wacan

      Cara gampang kanggo ngetik tandha wacan yaiku kanthi men-Swype saka tandha pitakon, koma, titik, utawa tandha wacan liyane menyang tombol Spasi, timbangane nutuli.

    • Milih Kabeh, Motong, Nyalin lan Nempel Pilih Kabeh: Swype saka menyang 'a'
      Potong: Swype saka menyang 'x'
      Salin: Swype saka menyang 'c'
      Tempel: Swype saka menyang 'v'
    • Trabasan aplikasi

      Google Maps: Swype saka menyang 'g', banjur 'm'

    • TelusuriSorot saperangan teks lan Swype saka menyang S kanggo nindakake panelusuran web cepet.
    • Ngalihake menyang basa sing mentas digunakake.Nalika nggunakake saperangan basa, cara sing paling cepet kanggo ngalihake mbalik menyang basa sadurunge yaiku kanthi men-Swype saka menyang tombol Spasi.
