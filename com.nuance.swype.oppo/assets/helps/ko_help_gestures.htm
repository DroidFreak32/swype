• 트레이스 동작

    트레이스 동작은 자주 사용하는 작업을 단축동작으로 빠르게 실행할 수 있게 해 줍니다.


    시스템 이용 서비스인 터치하여 탐색을 사용할 경우 이 기능은 이용하실 수 없습니다.


    • 편집 키보드 사용

      에서 특수문자 키(?123)로 트레이스하면 키보드 편집 메뉴가 표시됩니다.

    • 숫자 키보드 사용

      에서 숫자 5까지 트레이스하면 숫자 키보드를 바로 불러옵니다.

    • 키보드 감추기

      Swype 키에서 백스페이스 키로 트레이스하면 키보드를 쉽게 감출 수 있습니다.

    • 문장부호

      물음표, 쉼표, 마침표 등의 문장부호는 해당 부호에서 스페이스 키까지 트레이스하여 간단하게 입력할 수 있습니다.

    • 애플리케이션 바로 가기

      구글 지도: 부터 "g"로 다음에 "m"까지 트레이스 합니다.

    • 검색일부 텍스트를 하이라이트하고 부터 S까지 트레이스해서 빠른 웹 검색을 실시합니다.
    • 마지막 언어로 전환이 사용됐습니다.여러 언어를 사용할 때 이전 언어로 빨리 전환하는 방법은 에서 스페이스 키로 트레이스하는 것입니다.
