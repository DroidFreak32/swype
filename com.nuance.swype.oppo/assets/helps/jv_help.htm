Kanthi Swype, sampeyan bisa ngalihake patang mode input kanthi gampang - Swype, Ngomong, Nulis, utawa Nutul.

  • Swype

    Swype iku cara sing cepet kanggo ngetik teks. Swype bisa sampeyan gunakake kanggo ngetik tembung kanthi ngusap ngliwati aksara-aksara. Deleh driji sampeyan ana aksara sepisan saka tembung sing sampeyan karepake, lan gambar pola antar aksara, angkat driji sak uwise huruf terakhir. Swype bakal nyisipake spase yen dibutuhake.

    Pelajari luwih rinci
    • Tombol Swype

      Tombol Swype iku arupa tombol kanthi logo Swype. Penet lan tahan tombol Swype kanggo ngakses Setelan Swype.

      Tombol Swype uga digunakake kanggo ngawali saperangan isyarat Swype.

    • Isyarat Swype

      Isyarat Swype iku trabasan ana ing keyboard kanggo ngrampungi pakaryan kanthi cepet. Fitur iki ora kasedhiya yen sistem layanan aksesibilitas Explore-by-Touch mati.

      • Nggunakake Keyboard Edit Kanggo nggunakake Keyboard Edit, Swype saka menyang tombol Simbol (?123) ana ing keyboard.
      • Nggunakake keyboard angka Kanggo nggunakake Keyboard angka kanthi cepet, Swype saka menyang angka 5.
      • Ndhelikake keyboard Kanggo ndhelikake keyboard kanthi cepet, cukup Swype saka tombol Swype menyang tombol Backspace.
      • Mateni spasi otomatis Nonaktifake spasi otomatis sadurunge tembung sabanjure kanthi nge-Swype saka tombol Spasi menyang tombol Backspace.
      • Ngedit huruf gedhe-cilik tembung Owahi huruf gedhe-cilik tembung sauwise sampeyan ngetik kanthi nutul tembung, banjur nge-swype saka menyang tombol Shift . Dhaftar Pilihan Tembung kanthi opsi kapitalisasi bakal ditampilake, ngidini sampeyan kanggo milih aksara cilik, Dikapitalisasi, utawa KAPITAL KABEH.
      • Tandha wacan Cara gampang kanggo ngetik tandha wacan yaiku kanthi men-Swype saka tandha pitakon, koma, titik, utawa tandha wacan liyane menyang tombol Spasi, timbangane nutuli.
      • Milih Kabeh, Motong, Nyalin lan Nempel Pilih Kabeh: Swype saka menyang 'a'
        Potong: Swype saka menyang 'x'
        Salin: Swype saka menyang 'c'
        Tempel: Swype saka menyang 'v'
      • Trabasan aplikasiGoogle Maps: Swype saka menyang 'g', banjur 'm'
      • TelusuriSorot saperangan teks lan Swype saka menyang S kanggo nindakake panelusuran web cepet.
      • Ngalihake menyang basa sing mentas digunakake.Nalika nggunakake saperangan basa, cara sing paling cepet kanggo ngalihake mbalik menyang basa sadurunge yaiku kanthi men-Swype saka menyang tombol Spasi.
    • Ngetik aksara dobel

      Kanggo ningkatake akurasi nalika ngetik aksara dobel, coret bolak-balik utawa gawe bunderan ana ing aksara mau. Conto, kanggo nulis "gg" ing "inggih", coret bolak-balik ana ing tombol "g". Fitur iki ora kasedhiya yen sistem layanan aksesibilitas Explore-by-Touch mati.

    • Milih tembung

      Kanggo nampa tembung gawan sing disaranake ing Dhaftar Pilihan Tembung, terusake nge-Swype. Yen ora, tlusuri daftar kanthi nyeret driji sampeyan, bajur pilih tembung sing sampeyan karepake. Fitur iki ora kasedhiya yen sistem layanan aksesibilitas Explore-by-Touch mati.

      Kanggo milih teks saka dhaftar, obahake driji sampeyan mbunder kanggo ngrungokake entri dhaftar. Sak arah jarum jam bakal nlusuri dhaftar maju; nglawan arah jarum jam bakal nlusuri dhaftar mundur. Ngangkat driji bakal ngasilake entri dhaftar sing kaucap paling kari. Kanggo milih tembung saka Dhaftar Pilihan Tembung, geser munggah driji sampeyan saka keyboard nganti tembung kapisan ana ing dhaftar kaucap, banjur wiwiti gerakan mbunder. Fitur iki mung kasedhiya yen sistem layanan aksesibilitas Explore-by-Touch mati.

    • Ngapitalisasi tembung

      Fitur iki ora kasedhiya yen sistem layanan aksesibilitas Explore-by-Touch mati.

      Kapitalisasi tembung kanthi nge-Swype driji sampeyan saka huruf pertama ngliwati bagean dhuwur keyboard lan, tanpa ngangkat driji, geser driji mudhun menyang aksara sabanjure ing tembung iku mau.

      Owahi huruf gedhe-cilik tembung sauwise sampeyan ngetik kanthi nutul tembung, banjur nge-swype saka menyang tombol Shift . Dhaftar Pilihan Tembung kanthi opsi kapitalisasi bakal ditampilake, ngidini sampeyan kanggo milih aksara cilik, Dikapitalisasi, utawa KAPITAL KABEH.

      Kanggo ngetik ing mode CAPS Lock, cukup tutul ping pindho ana ing tombol Shift .

    • Spasi otomatis

      Swype otomatis nyisipake spasi antarane tembung nalika sampeyan ny-Swype tembung sabajure ing ukara mau. Sampeyan bisa nguripi utawa mateni fitur spasi otomatis ing Setelan Swype.

      Spasi otomatis bisa dipateni kanggo sak tembung kanthi nge-Swype saka tombol spasi menyang tombol Backspace.

      Fitur iki ora kasedhiya yen sistem layanan aksesibilitas Explore-by-Touch mati.

    • Ngganti tembung

      Ganti tembung kathi nutul ana ing tembung iku mau, banjur pilih tembung sing sampeyan karepake saka Dhaftar Pilihan Tembung, utawa cukup sorot tembung lan Swype tembung anyar. Tembung anyar bakal ngganti tembung sing salah.

      Tembung bisa disorot kanthi nutul tembung lan nyenggol utawa nutul pindho tembung mau.

    • Mendhal antarane aksara

      Kadang ngendhani aksara nalika nge-Swype bisa njamin sampeyan bisa entuk tembung sing sampeyan karepake dhisik.

      Contone, "tipi" lan "topi" bisa nggambar kanthi pola sing padha - ananging, sampeyan kudu mindhah antarane aksara ing garis lurus. Ngendhani "o" nalika nge-swype driji sampeyan menyang "p" bisa njamin menawa tembung "tipi" iku sing sepisanan ing Dhaftar Pilihan Tembung.

    • Karakter alternatif

      Penet lan tahan ana ing salah siji tombol kanggo ngatonake dhaftar karakter alternatif kanggo tombol iku mau, contone aksara beraksen kayata é lan è, simbol kayata @ lan %, sarta angka.

      Tutul tombol Simbol (?123) kanggo mbukak Keyboard simbol.

      Wigati dimangerteni, KABEH karakter bisa di-Swype saka keyboard utama (bisa sampeyan deleng utawa ora). Sampeyan bisa men-Swype nggunakake tampilan keyboard, ananging sampeyan mung oleh tembung sing nduweni paling ora siji angka utawa simbol.

    • Nambah lan mbusak tembung

      Swype kanthi wasis nambah tembung anyar sing sampeyan gunakake menyang Kamus Personal.

      Sampeyan uga bisa nambah tembung kanthi nyorot tembung iku mau, banjur nutul . Tutul pituduh sing katon kanggo nambahake tembung.

      Kanggo mbusak tembung, penet lan tahan tembung ing Dhaftar Pilihan Tembung, banjur tutul Oke ing dialog konfirmasi. Fitur iki ora kasedhiya yen sistem layanan aksesibilitas Explore-by-Touch mati.

  • Ngomong

    Sampeyan bisa ngomong kanggo ngetik konten teks kanggo apa wae, mulai saka ngirim pesen teks lan pesen email nganti update Facebook lan Twitter.

    Pelajari luwih rinci
    • Tandha wacan

      Ora perlu nambahake tandha wacan kanthi manual. Cukup ucapake tandha wacan sing sampeyan karepake lan lanjut. Coba iki:

      1. Penet tombol swara lan mulai ngomong.
      2. Apa sing sampeyan ucapake: Mangan wengi lezat tanda seru
      3. Apa sing sampeyan olehi: Mangan wengi lezat!

    • Input swara ora kasedhiya ana ing saperangan keyboard

  • Tulis

    Sampeyan uga bisa nggunakake driji kanggo nggambar aksara lan tembung, Swype uga bakal ngubah menyang teks. Sampeyan bisa nggambar aksara saka kiwa menyang tengen utawa ana ing bagean ndhuwure. Penet ABC / 123 kanggo ngalihake antarane mode aksara lan simbol.

    Pelajari luwih rinci
    • Aktifake tulisan tangan
      1. Penet lan tahan lan geser driji sampeyan menyang ikon tulisan tangan.
      2. Gambar aksara nganggo driji ana ing area tulisan tangan.
      3. Tutul spasi antarane saben tembung
    • Isyarat Multi-Demok

      Isyarat multi-demok mbantu sampeyan nglengkapi tugas sederhana, kayata ngapitalisasi tembung lan aksara.

      1. Gambar saperangan aksara cilik ana ing papan gambar
      2. Sak uwise ngetik karakter-karakter mau, geser rong driji munggah ana ing area tulis
      3. Fitur tulisan tangan bakal ngenali isyarat multi-demok lan ngapitalisasi aksara

    • Tulisan tangan kasedhiya ana ing saperangan keyboard.

      Fitur iki ora kasedhiya yen sistem layanan aksesibilitas Explore-by-Touch mati.

  • Ketik

    Wujud tradisional input keyboard manual. Tutul input ana ing keyboard Swype digawe luwih gampang lan luwih efisien kanthi saperangan fitur sing banget mbantu:

    Pelajari luwih rinci
    • Koreksi Ketikan Ora Rapi

      Sampeyan ora kudu nutul saben-saben aksara kanti sempurna. Cukup usahakne saapike lan Swype bakal kanthi wasis menehi usulan tembung.

    • Ngrampungake Tembung

      Swype uga bakal mrakirakake tembung nalika sampeyan nutul mung saperangan aksara.

  • Basa

    Kanggo ngalihake basa saka keyboard: Penet lan tahan tombol spasi. Pilih basa sing sampeyan karepake saka menu popup.

  • Swype Connect

    Kanthi Swype Connect, sampeyan bisa ngirimake update lan fungsionalitas hebat langsung saka perangkat sampeyan! Amarga Swype Connect nggunakake 3G, murih apike gunakake koneksi WiFi

    Pelajari luwih rinci
    • Undhuhan Basa

      Nambah basa tambahan menyang Swype iku gampang:

      1. Penet lan tahan lan pilih Basa.
      2. Saka menu Basa, pilih Undhuh basa.
      3. Klik basa lan undhuhan sampeyan bakal miwiti kanthi otomatis.

    • Swype Connect ora kasedhiya ana ing kabeh keyboard.
