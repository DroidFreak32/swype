• Swype-Gesten

    Swype-Gesten sind Tastaturkürzel, mit denen Sie häufige Aufgaben schneller erledigen.


    Diese Funktion ist nicht verfügbar, wenn der Systemzugänglichkeitsdienst Explore-by-Touch aktiviert ist.


    • Öffnen der Bearbeitungstastatur

      Um die Bearbeitungstastatur anzuzeigen, swypen Sie von zur Symbol-Taste (?123) auf der Tastatur.

    • Öffnen des Ziffernblocks

      Um den Ziffernblock schnell anzuzeigen, swypen Sie von zur Ziffer 5.

    • Tastatur ausblenden

      Um die Tastatur auszublenden, swypen Sie von der Swype-Taste zur Rückschritt-Taste.

    • Automatisches Einfügen von Leerstellen deaktivieren

      Das automatische Einfügen von Leerstellen vor dem nächsten Wort kann deaktiviert werden, indem Sie von der Leertaste zur Rückschritt-Taste swypen.

    • Wortauswahl ändern

      Um die Groß-/Kleinschreibung eines Wortes nach seiner Eingabe zu ändern, tippen Sie auf das Wort im Text und swypen Sie von der -Taste zur Umschalttaste . Eine Wortauswahl-Liste mit Optionen zur Groß-/Kleinschreibung wird angezeigt, in der Sie Kleinschreibung, Großschreibung oder ALLES GROSS auswählen können.

    • Satzzeichen

      Eine einfache Möglichkeit zum Eingeben von Satzzeichen besteht darin, vom Fragezeichen, Komma, Punkt oder anderen Satzzeichen zur Leertaste zu swypen, anstatt auf das Satzzeichen und dann die Leertaste zu tippen.

    • Alles auswählen, Ausschneiden, Kopieren und Einfügen Alles auswählen: Swypen Sie von zu „a“.
      Ausschneiden: Swypen Sie von zu „x“.
      Kopieren: Swypen Sie von zu „c“.
      Einfügen: Swypen Sie von zu „v“.
    • Anwendungs-Tastaturkürzel

      Google Maps: Swypen Sie von zu „g“ und dann „m“.

    • SucheMarkieren Sie einen Text und swypen Sie von zu S, um eine schnelle Websuche zu starten.
    • Auf die letzte verwendete Sprache umschalten.Wenn Sie mehrere Sprachen verwenden, können Sie schnell durch Swypen von zur Leertaste auf die vorherige Sprache umschalten.
