• Gestos Swype

    Gestos Swype são atalhos no teclado para realizar rapidamente as tarefas simples.


    Esta função não está disponível quando o serviço de acessibilidade do sistema Explore-by-Touch está ativado.


    • Visualizando o Teclado de Edição

      Para visualizar o teclado de Edição, trace desde o até a tecla de Símbolos (?123) do teclado.

    • Visualizar o Teclado Numérico

      Para acessar rapidamente o teclado numérico, trace desde o até o número 5.

    • Ocultando o Teclado

      Para ocultar facilmente o teclado, basta traçar a partir da tecla Swype até a tecla backspace.

    • Desligar o Espaçamento Automático

      Remova o espaçamento automático antes da próxima palavra deslizando desde a tecla de espaço até a tecla Backspace.

    • Editar a Caixa de uma Palavra

      Altere a formatação de uma palavra para maiúscula/minúscula após inseri-la tocando na palavra e em seguida deslizando a partir do até a tecla Shift . Uma Lista de Palavras Selecionadas com opções de letras maiúsculas será exibida, permitindo a você escolher entre as formatações minúscula, primeira letra maiúscula ou colocar TODAS AS LETRAS EM MAIÚSCULAS.

    • Pontuação

      Uma forma simples de inserir a pontuação é traçar desde o símbolo de interrogação, vírgula, ponto final ou outra pontuação até a tecla de espaço, em vez de tocá-los individualmente.

    • Selecionar Tudo, Cortar, Copiar e Colar Selecionar Tudo: Trace a partir da tecla até a tecla ‘a’
      Cortar: Trace a partir da tecla até a tecla ‘x’
      Copiar: Trace a partir da tecla até a tecla ‘c’
      Colar: Trace a partir da tecla até a tecla ‘v’
    • Atalhos de Aplicativos

      Google Maps: Trace desde a tecla até à ‘g’ e em seguida ‘m’

    • ProcurarRealce o texto pretendido e Swype do ao S para efectuar uma pesquisa rápida na Web.
    • A muda para o último idioma utilizado.Se utilizar vários idiomas, a forma mais rápida de mudar novamente para o idioma anterior é deslizar a partir do para a tecla de espaço.
