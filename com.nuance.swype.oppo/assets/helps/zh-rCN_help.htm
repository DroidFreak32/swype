通过 Swype，您可以在四种不同的输入模式之间无缝地切换 – Swype（滑行）、Speak（说）、Write（写）或 Tap（轻击）。

  • Swype

    Swype 是一種輸入文字的快速方式。 它可以帮您通过在字母上划线输入字词。 将您的手指放在字/词的第一个字母上，然后划过每个字母，到最后一个字母后放开。 软件会在需要时自动插入空格。

    • Swype 键

      “设置”键是带有 Gear 标志的按键。 长按 Swype 键可访问 Swype 设置。

      Swype 键还可用于加入许多 Swype 手势。

    • Swype 手势

      Swype 手势是在键盘上快速完成常用任务的快捷方式。 系统辅助服务 Explore-by-Touch 开启时，此功能无法提供。

      • 切换到编辑键盘 要切换到编辑键盘，可在键盘上从 滑行到符号键（?123）。
      • 切换到数字键盘 如要快速切换到数字键盘，可从 滑行到数字 5。
      • 隐藏键盘 只需从 Swype 键滑行到退格键，即可轻松地隐藏键盘。
      • 标点 有一种输入标点符号的简单方式是从问号、逗号、句号或其它标点符号滑行到空格键，而不是轻击它。
      • 全选、剪切、复制和粘贴 全选： 从 滑行至 ‘a’ 键
        剪切： 从 滑行至 ‘x’ 键
        复制： 从 滑行至 ‘c’ 键
        粘贴： 从 滑行至 ‘v’ 键
      • 应用程序快捷键Google 地图： 从 滑行至 ‘g’ 键，然后到 ‘m’ 键
      • 搜索高亮显示一些文本并从 滑行（Swype ）至 ‘S’ 键，进行快速网络搜索。
      • Swype 提供了最后使用的语言。使用多国语言时，可以从 滑动（Swype ）到空格键，快速切换到先前使用的语言。
    • 输入双重字母

      要提高输入重复字母时的精确度，可在字母上轻划或划圈。 例如，在「n」键上轻划可以拼出「HenNan（很难）」中的「nn」。 系统辅助服务 Explore-by-Touch 开启时，此功能无法提供。

    • 选择字词

      如果接受字词选择列表中建议的默认字词，只需继续以 Swype 方式输入。 否則，請拖曳您的手指來捲動清單，並選擇您想要的字詞。 系统辅助服务 Explore-by-Touch 开启时，此功能无法提供。

      系统辅助服务 Explore-by-Touch 开启时，此功能才可提供。

    • 在字母之间跳动

      有时在跟踪输入时避开字母，可确保您第一次就能找到您想要的字词。

      例如：「rui（瑞）」和「ri（日）」所划过的字母路线可能一样，但是请注意，划过各个字母时不需要是一条直线。 当手指滑动到 "i" 时，避开 "u" 可确保“ri（日）”显示在字词选择列表的第一位。

    • 备选字符

      长按一个键可弹出该键的供选字符列表，例如 % 和 @ 等符号以及数字

      轻击一下符号键 (?123) 可进入符号键盘。

      请注意所有的字符均可用 Swype 方式从主键盘输入（无论您是否能看见这些字符）。 您可以使用这一视图的键盘以跟踪方式输入，但您只能得到至少有一个数字或符号的单词。

    • 添加和删除字词

      Swype 可智能地添加您使用的新字词到您的 Personal Dictionary（个人字典）。

      您还可以通过高亮一个字词并轻击 来添加该字词。 轻点一下所出现的提示，以新增字词。

      要删除字词，在字词选择列表中长按该字词，然后在确认对话框中轻击“OK（确定）”。 系统辅助服务 Explore-by-Touch 开启时，此功能无法提供。

    • 个性化

      Swype可快速将Facebook、Twitter、Gmail中的字词添加到您的字典。 个性化 Swype：

      1. 长按
      2. 从 Swype 设置菜单，选择个性化设置
      3. 从个性化选项中选择，如果提示的话，输入您的证书。
      4. 您可以从一个或所有来源个性化 Swype。
  • 说

    您可以通过说的方式输入一切文字内容，从文本和电子邮件消息到 Facebook 和 Twitter 更新。

    取消
    • 标点

      您无须手动添加标点符号。 只需说出您要的标点符号然后继续。 我们来试一下：

      1. 按语音键 然后开始说话。
      2. 您说的内容： 晚餐太美味了感叹号
      3. 您得到的内容： 晚餐太美味了！

    • 有些键盘上没有语音输入功能

  • 写

    您可以用手指画出字母和字词，Swype 将把它转换成文字。 您可以从左到右或在彼此的上方画出字母。

    取消
    • 启用手写模式
      1. 长按 并将手指滑向手写图标。
      2. 用手指在手写区画出字母。

    • 有些键盘上没有手写功能。

      系统辅助服务 Explore-by-Touch 开启时，此功能无法提供。

  • 点击

    手动键盘输入的传统形式。 通过一些有帮助的功能可以使在 Swype 键盘上轻击输入变得更加简单高效：

    取消
    • 潦草输入更正

      您无须完全正确地轻击每一个字母。 您只需尽力即可，Swype 将智能地提供字词建议。

    • 单词自动完成

      当您只点了少数字母时，Swype 还能猜测您想要的字词。

  • 语言

    从键盘切换语言： 长按空格键。 从弹出的菜单中选择您的目标语言。

    取消
    • 切换输入模式

      切换键盘布局

    • 切换键盘手写和滑动输入

      想要在全键盘上切换键盘手写和滑动输入种类，只需按一下功能表上的手写图标。

    • 中英文混输

      中文拼音全键盘输入支持中英文混输。 候选词可以作为英文输入供用户选择。如果未在当前选择条中查找到英文词，请通过单击选择条右边的按键， 或向左拖动选择条查找。

    • 切换键盘布局

      Swype为不同的输入模式提供多种键盘布局，输入模式有中文拼音，注音，手写，仓颉，快速仓颉。

      在键盘输入时，使用两个手指从左向右或者从右向左滑动，您可以在数字键盘和全键盘间来回切换。

      在手写识别区域，使用两个手指向上或向下滑动，您可以在全屏模式和半屏模式间进行快速切换。

    • 关闭输入键盘

      您可以使用两个手指在输入区域内向下滑动，即可快速关闭输入键盘。
