• Gestos Swype

    Gestos Swype são atalhos no teclado para realizar rapidamente as tarefas simples.


    Este recurso não está disponível quando o serviço de acessibilidade do sistema Explore-by-Touch estiver ligado.


    • Visualizando o Teclado de Edição

      Para visualizar o teclado de Edição, deslize o dedo desde o até a tecla de Símbolos (?123) do teclado.

    • Visualizar o Teclado Numérico

      Para acessar rapidamente o teclado numérico, deslize o dedo desde o até o número 5.

    • Ocultando o teclado

      Para ocultar facilmente o teclado, basta deslizar o dedo a partir da tecla Swype até a tecla backspace.

    • Desligar o espaçamento automático

      Remova o espaçamento automático antes da próxima palavra deslizando desde a tecla de espaço até a tecla Backspace.

    • Editar a Caixa de uma Palavra

      Altere a formatação de uma palavra para maiúscula/minúscula após inseri-la tocando na palavra e em seguida deslizando o dedo a partir do até a tecla Shift . Será exibida uma Lista de seleção de palavras com opções de maiúsculas, permitindo-lhe escolher minúscula, Primeira letra maiúscula ou TODAS MAIÚSCULAS.

    • Pontuação

      Uma forma simples de inserir a pontuação é traçar desde o símbolo de interrogação, vírgula, ponto final ou outra pontuação até a tecla de espaço, em vez de tocá-los idividualmente.

    • Selecionar Tudo, Cortar, Copiar e Colar Selecionar Tudo: Deslize o dedo a partir da tecla até a tecla ‘a’
      Cortar: Deslize o dedo a partir da tecla até a tecla ‘x’
      Copiar: Deslize o dedo a partir da tecla até a tecla ‘c’
      Colar: Deslize o dedo a partir da tecla até a tecla ‘v’
    • Atalhos de Aplicativos

      Google Maps: Deslize o dedo desde a tecla até a ‘g’ e em seguida ‘m’

    • BuscarRealce o texto pretendido e deslize o dedo do ao S para efetuar uma pesquisa rápida na Web.
    • Alternando para o último idioma utilizado.Quando estiver utilizando diversos idiomas, a forma mais rápida de alternar novamente para o idioma anterior é deslizar o dedo a partir do para a tecla espaço.
